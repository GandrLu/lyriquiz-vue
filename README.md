# Lyriquiz

This is Lyriquiz, a lyric based song quiz. It uses the spotify and the lrclib api to generate personalised questions.  
Play it here:  
https://lyriquiz-2aea0.web.app/  
(requires a spotify account)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
